# Global Electronics Sales & Supervisor Performance Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze and visualize global electronics sales data. The dashboard provides detailed insights into sales performance across different product categories, brands, states, and supervisors.

Users can dynamically filter the dashboard by selecting a supervisor to analyze their individual sales performance and explore sales trends across multiple brands and regions worldwide.

---

## 🚀 Features
- Interactive sales analytics dashboard
- Supervisor-wise sales performance tracking
- Worldwide sales analysis by state/location
- Brand-wise sales distribution visualization
- Product category and order status analysis
- Dynamic filtering and slicers for better insights
- KPI-driven business intelligence reporting
